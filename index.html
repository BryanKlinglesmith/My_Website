Welcome!

Hello! My name is Bryan. Welcome to my website! Let me tell you a little about myself. For the past 20+ years, I have machinist. I have spent most of that time as a CNC Programmer and operator. I have run both CNC Lathes and CNC Mills during my career so far. I am currently running a 7 axis CNC Lathe with duel spindles. It is a complicated machine but I enjoy the challenge of programming this machine. I am proud of the fact that I mostly trained myself to program and run this machine.

In January 2018, I began learning to program. I started the Code Louisville Front End Web Development program. I am learning to code in HTML5, CSS, JavaScript, Sass and Angular. I have almost reached the end of the first 12 week session and will be able to call myself a junior Front End Web Developer. It has been a dream of mine to be a computer programmer ever since I was in high school. With a little luck and a lot of hard work, I am getting closer to making this dream a reality.
