Welcome!

When I was a teenager , my dad bought us our families first PC. I had used computers at school but we had never owned our own home computer. It had a DOS interface on it and it was the most amazing thing I had ever seen! I was fascinated with every aspect of the PC. I was always doing something on it just to see how it worked.

I have spent most of my life tinkering with, and trying to figure out how computers work. I took my first programming class in high school. I was taught how to program in BASIC.

In January 2018, I began learning to program. I started the Code Louisville Front End Web Development program. I am learning to code in HTML5, CSS and JavaScript. By the end of the Code Louisville program I will be a Full Stack junior developer.

I have spent the last 22 years working as a machinist. I program and operate CNC mills and lathes. I have been programming CNC machines for most of my career. I am currently running a 7 axis CNC Lathe with duel spindles. It is a complicated machine but I enjoy the challenge of programming this machine. I am proud of the fact that I mostly trained myself to program and run this machine.
